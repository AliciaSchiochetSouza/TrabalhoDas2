# ADR-003 — Escolha da Biblioteca de Acesso ao Banco de Dados

## Status

Aprovado

## Contexto

Foi realizada uma PoC em Azure Functions para comparar o desempenho das bibliotecas **PyODBC** e **SQLAlchemy** em consultas ao SQL Server.

Os testes utilizaram a mesma consulta e foram executados duas vezes para obtenção da média dos tempos.

## Decisão

Foi escolhido o **PyODBC**, pois apresentou melhor desempenho nos testes realizados.

## Opções Consideradas

### PyODBC

* Melhor tempo de execução;
* Conexão direta com o SQL Server.

### SQLAlchemy

* Maior abstração e facilidade de manutenção;
* Desempenho inferior no cenário testado.

## Consequências

### Positivas

* Consultas mais rápidas;
* Menor tempo de resposta da aplicação.

### Negativas

* Menor nível de abstração;
* Mais código SQL manual.
